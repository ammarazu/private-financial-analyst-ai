# Private Financial Analyst AI
### Built by Ammara Zubair | Noufal Digital LLC

## What This System Does
A fully private, secure AI system for financial document analysis.
Upload your financial documents. Ask complex questions. Get accurate
answers with source citations. Your data never leaves your server.

## Who This Is For
- Wealth management firms
- Investment analysts  
- Private equity teams
- Accounting and audit firms

## Security Features
- PII detection and removal via Microsoft Presidio
- Prompt injection protection via LLM Guard
- Private LLM deployment via AWS Bedrock or Ollama
- Zero data sent to third-party APIs
- All secrets managed via AWS Secrets Manager
- Infrastructure as code via Terraform

## System Components
- Document Parsing: Unstructured.io
- Vector Database: Weaviate (hybrid search)
- Embeddings: HuggingFace (local, free)
- Reranking: Cohere Rerank
- LLM Router: LiteLLM
- Private LLM: AWS Bedrock / Ollama
- UI: Streamlit
- Cost Tracking: Langfuse
- Quality Evaluation: Ragas
- Observability: Arize Phoenix
- Infrastructure: Terraform + AWS

## Documents Supported
- PDF annual reports
- SEC filings (10-K, 10-Q)
- Earnings call transcripts
- Market research reports
- Financial statements

## Evaluation Metrics
- Faithfulness: > 0.75
- Answer Relevancy: > 0.80
- Context Precision: > 0.70

## Deployment
Single command deployment via Terraform.
Full system operational in under 30 minutes.

## Pricing
- Setup fee: $3,000 - $8,000
- Monthly maintenance: $500 - $1,500
