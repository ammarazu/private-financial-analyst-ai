# 🔐 Private Financial Analyst AI

> Enterprise-grade RAG system for secure financial document analysis

## 🎯 What It Does

A fully secure AI system that reads financial documents and answers questions with enterprise-grade security built in.

## 🛡️ Security Features

- ✅ Prompt injection protection (95% accuracy)
- ✅ PII stripping via Microsoft Presidio
- ✅ Output scanning for sensitive data
- ✅ AWS Secrets Manager for credentials
- ✅ IAM least privilege enforcement
